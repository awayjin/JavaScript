  # 前端性能提升
  ## 1.0 服务器方面(server):
  - 使用cdn
  - 减少不必要的数据返回
  - 使用gzip
  - 缓存( etag/expires ...)

  ## 2.0 内容方面(content)
  - 减少 http 请求(css sprites)
  - 不同资源放在不同的域下(http1.1)
  - 2.3 预加载(preload)/延迟加载/延迟执行(立即下载，延迟执行[before DOMContentLoaded]defer)
    - async 异步执行该脚本
    - defer 这个布尔属性被设定用来通知浏览器该脚本将在文档完成解析后，触发DOMContentLoaded事件前执行

  - 精简 HTML 结构
  - 压缩资源

  ## 3.0 css
  - in head
  - 较少的层级

  ## 4.0 JS
  - 减少DOM访问

  ## 5.0 Webapck
  - tree shaking 去除没有使用的代码
  - 提取公共包，有被问到
  - 拆分模块，按需加载
  - 优化图片，使用 base64 代替小图
  - file name with hash (etag)
