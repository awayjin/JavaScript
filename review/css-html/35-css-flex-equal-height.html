flex 属性 两列等高 CSS属性 flex 规定了弹性元素如何伸长或缩短以适应flex容器中的可用空间。 这是一个简写属性，用来设置 flex-grow, flex-shrink 与 flex-basis。 
    justify-content 主轴对齐

    align-items 侧轴上项目对齐方式
    align-content 堆栈伸缩行

    flex 伸缩性
    
      align-self 侧轴上单个项目对齐方式

      flex-flow 伸缩流的方向与换行
      flex-shrink 伸缩-收缩比率
      flex-basis 伸缩-基准值

      flex-direction 伸缩流方向

      flex-grow 伸缩-扩展基数
      flex-wrap 伸缩-换行

      order 伸缩-顺序
