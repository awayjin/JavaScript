## 4.4 保护数据结构内部元素
ES2015 类是基于原型，这种方式不能声明私有属性(变量)或 方法
```javascript
class Stack {
  constructor () {
    this.items = [11, 22]
  }
}
const stack = new Stack()
// 可访问可改变
console.log(Object.getOwnPropertyNames(stack))
console.log(Object.keys(stack)); // {2}
console.log(stack.items); // {3}

```
