# 继承
> 被继承的类叫超类(superclass),继承超类的类叫子类(subclass)

### 8.1.1 成员的访问和继承
> 尽管子类包括超类的所有成员，它不能访问超类被声明private的成员

### 8.1.2 更实际的例子
> 继承的一个主要优势一旦你已经创建了一个超类，而该超类定义了适用于一组对象的属性，它可用来创建任何数量的说明更多细节的子类
