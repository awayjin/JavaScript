## GraphQL
GraphQL 是一个用于 API 的查询语言，是一个使用基于类型系统来执行查询的服务端运行时（类型系统由你的数据定义）。


API 服务 - GraphQL:
- 专注数据聚合，前端要什么就返回什么
- Facebook 开发的实现 API 服务的库
  - 让前端有“自定义查询”数据的能力
  
- koa-graphql
