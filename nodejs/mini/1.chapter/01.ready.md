## 1-6 安装 Node.js v8.0 生产环境
sudo apt-get update

nvm 安装多版本 node

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.1/install.sh | bash

nvm ls 查看版本

安装最新版本
nvm install v13.3.0

切换版本
nvm use v13.3.0

设置默认版本：
nvm alias default v13.3.0
